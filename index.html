TDP FEATURES

DESIGN METHODS

Techgrid Designer Pro is equipped to design reinforced soil walls using three of the most widely used methodologiest:

BS 8006-1:2010

Analysis has been done accordance to BS 8006-1:2010 standard , for static condition. There are various cases,to define worst condition for rupture,pullout and servicability checks. Seismic conditons are checked as per FHWA guidelines.

AASHTO LRFD

Analysis has been done in accordance with FHWA guidelines (AASHTO), for static and seismic conditions. All the necessary checks are carried out to satisfy the minimum factor of safety, as per AASHTO standard.

NCMA

Analysis has been done accordance to National Concrete Masonry Association standard, for Segmental Block Wall. All the checks, including friction connection, is carried out as per NCMA guidelines.
